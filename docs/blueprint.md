# **App Name**: SwapCircle

## Core Features:

- User Authentication: Secure login/signup with email, password, and Google authentication. Includes forgot password functionality.
- Item Listing: Users can upload items for swapping with details like name, category, condition, and description.
- Item Discovery: Browse and search items, filter by location, condition, and category.
- Trade Request: Request trades by selecting items to offer in exchange.
- Trade Chat: Real-time chat functionality for negotiating trades.
- User Profiles: View user profiles with listings, swap history, and ratings.
- Homepage: A professional homepage with a large hero section featuring an image of people swapping items, a catchy headline, a brief description, and two call-to-action buttons: 'Browse Items' and 'List an Item'. The section will also include a smaller image showcasing a popular item on the right side.

## Style Guidelines:

- Primary color: Soft pastel blue (#A7D9ED) to create a warm and trusting environment.
- Background color: Very light blue (#F0F8FF) to keep a clean and simple background.
- Accent color: Pale orange (#F9C784) for buttons and highlights, offering a touch of warmth.
- Font: 'Inter', sans-serif, for a modern and clean interface (as per user request).
- Light and friendly icons with rounded corners.
- Modern, clean layout with rounded corners and soft shadows, generous spacing, and minimalistic design.
- Smooth transitions and subtle animations on item cards.
